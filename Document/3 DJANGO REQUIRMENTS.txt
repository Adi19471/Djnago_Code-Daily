        HOW TO INSTALL DJANGO 

        * Separate Virtuval Enviralment
        * GLobal inveiralment


        INSTALL DJANGO in SEPARATE Environment
        Install Virtuval Enviralment Wrapper

        pip install virtualenvwrapper-win -------> This is used to install Virtuval Enviralment Wrapper


        Create VIrtuval Enviralment(VE)

        mkvirtualenv envname -This is used virtuval envirolment .It will automatically activated envirolment


        Active VE
        workon envname - this is used activate envirolment


        INSTAL DJANGO in Created VE

        pip install django -First activate envirolment then the run the command to install django within activate envirolment.You can specify pip install django==2.0
